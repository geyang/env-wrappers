# `gym-wrappers`, a collection of wrappers for OpenAI Gym environments

Current release at `v0.1.0`.

## Installation

To install:
```bash
pip install gym-wrappers
```
or
```bash
pip install git+https://github.com/geyang/gym-wrappers
```

## Usage Example

```python
import gym
from gym_wrappers import VecEnv

# your code here...
```
